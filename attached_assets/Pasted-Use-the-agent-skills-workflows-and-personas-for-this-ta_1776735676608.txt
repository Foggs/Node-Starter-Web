## Task: Add rewrite variety to the Improved Replay

## Problem

Every improved manager turn currently opens with the same sentence
structure — "I see that you're...", "I can see that...", "I want
to acknowledge that..." — regardless of turn number, scenario,
persona, or what the employee just said. After 5 turns, the
rewrites feel templated and robotic, which directly undermines the
product's core promise of sounding like a better version of YOU.

## Incremental implementation plan (follow this order)

**Slice 1 — Audit the current rewrite prompt**
- Find the GPT-4o-mini system prompt used in
  POST /api/improved-replay
- Print the full prompt as-is — do not change it
- Identify exactly what instruction (or lack of instruction)
  is causing the repetitive openers
- Document the finding as a code comment before proceeding

**Slice 2 — Define the variety system**

Add the following rules to the rewrite system prompt:

TURN-BY-TURN OPENER RULES:

Turn 1: Lead with the purpose of the meeting. Be direct.
  Do not open with empathy or acknowledgment of feelings.
  Register: "I want to be direct with you about..."

Turn 2: Respond to what the employee said. Acknowledge
  their specific point before redirecting.
  Register: "I hear that you feel... and I also need us
  to be clear about..."

Turn 3: Hold the position. Open with a calm, firm restatement.
  Do not re-litigate the decision.
  Register: "This decision is final. I want to focus on
  what happens next..."

Turn 4: Move toward resolution. Bridge from difficulty
  to the path forward.
  Register: "I know this is hard to hear. Here is what
  support looks like from here..."

Turn 5: Close with clarity and dignity. Open with a
  forward-looking statement. Give the employee something
  concrete to hold onto. Do not end with a question.
  Register: "Before we finish, I want to make sure you
  are clear on next steps..."

GLOBAL RULES (all turns):
- Never open two consecutive turns with the same first 3 words
- Never use these openers in any turn:
    "I see that"
    "I can see that"
    "I understand that"
    "I want to acknowledge"
- Vary sentence length across turns
- Match emotional register to persona:
    tearful      → warmer tone, shorter sentences, slower pacing
    defensive    → calm, firm, non-reactive, no hedging language
    angry        → de-escalating, steady, no mirroring of hostility
    withdrawn    → gentle, direct, fill silence with clarity
    professional → peer-to-peer tone, efficient, respectful

**Slice 3 — Inject turn_index and persona into the prompt**

The rewrite prompt must receive:
- turn_index (1–5) to apply the correct opener rule
- persona to apply the correct emotional register
- employee_transcript for that turn so the rewrite responds
  to what was actually said

Confirm these values are available where the prompt is built.
If not, trace them through session state and thread them in.

**Slice 4 — Run the full test suite**
